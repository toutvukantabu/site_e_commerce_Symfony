#  e-commerce Website

after cloning , open the terminal at the root of the project (./site_e_commerce_Symfony) : 

``` 
make build
``` 
and next 

``` 
make start
``` 
If you need help, see "MAKEFILE" or type

``` 
make help
``` 
to see the list of commands in terminal
